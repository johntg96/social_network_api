# Social Network API  
  
## Description  
A backend API for a social networking type of site built with MongoDB and Express. 
Video Demonstration available [here](https://drive.google.com/file/d/1-Sb0aqPCoGJ9OyiJzI8oAm6il2B9351o/view)   
  
## Installation  
`npm i`
  
## Usage  
`npm run start`
